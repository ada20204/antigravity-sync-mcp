## 使用说明

- 这些项目用于调研与方案对比，不建议直接混合拷贝到生产代码。
- 更新引用版本请使用 submodule 流程维护，并记录变更原因。
- 若某个参考项目未初始化，可执行：

```bash
git submodule update --init --recursive
```
